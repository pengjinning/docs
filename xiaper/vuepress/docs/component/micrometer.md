# Micrometer

::: tip 提示
[Micrometer](http://micrometer.io/) is a metrics instrumentation library for JVM-based applications.
[Prometheus](https://prometheus.io/) is an open source tool for instrumenting and monitoring metrics
[Grafana](https://grafana.com/) is an analytics platform for your metrics, it allows you to query, visualize and alert on your metrics
:::
